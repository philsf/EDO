\subsection{Derivadas}

Quando existe, a derivada de uma função $f(x)$ no ponto $x_0$ é o valor $f'(x_0)$ que
corresponde à variação instantânea de $f$ no ponto $x_0$. Ao variar o
ponto $x_0$ para outro ponto $x_1$, obtemos o valor $f'(x_1)$,
possivelmente diferente. Assim sucessivamente, se percorrermos todos
os valores possíveis para
